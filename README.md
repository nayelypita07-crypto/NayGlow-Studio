Nombre del Proyecto: 
NayGlow Studio

Descripción de la Idea: 
NayGlow Studio es una aplicación web responsive enfocada en la presentación de servicios de belleza y estética. La página fue diseñada para mostrar de manera organizada e interactiva los servicios ofrecidos, como pintado de uñas, maquillaje profesional, depilación y laminado de cejas.

La aplicación permite al usuario navegar fácilmente entre las diferentes secciones mediante una interfaz moderna, visual y adaptable a distintos dispositivos.

Problema que Resuelve: 
Actualmente muchos pequeños negocios de belleza no cuentan con una plataforma digital moderna donde puedan mostrar sus servicios de manera organizada y atractiva. Esto dificulta que los clientes conozcan la información del negocio y los servicios disponibles.

NayGlow Studio busca solucionar este problema mediante una aplicación web visual e interactiva que permita presentar los servicios de belleza de forma clara, moderna y accesible desde cualquier dispositivo.

Público Objetivo
- El proyecto está dirigido principalmente a:
- Personas interesadas en servicios de belleza y estética.
- Clientes que buscan información sobre maquillaje, uñas y cuidado personal.
- Emprendimientos o salones de belleza que desean mejorar su presencia digital.

Framework Seleccionado
El framework seleccionado para el desarrollo de la aplicación fue:
- React JS
Además, el proyecto fue creado utilizando Vite como herramienta de desarrollo.

Tecnologías Utilizadas
Las tecnologías utilizadas en el desarrollo del proyecto fueron:
- React JS
- Vite
- JavaScript
- CSS
- React Router DOM
- HTML5

Pasos de Instalación o Ejecución

Crear el proyecto
npm create vite@latest

Ingresar a la carpeta del proyecto
cd nayglow-studio

Instalar dependencias
npm install

Instalar React Router DOM
npm install react-router-dom

Ejecutar el proyecto
npm run dev
